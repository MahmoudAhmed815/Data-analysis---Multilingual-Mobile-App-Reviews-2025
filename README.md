# Mobile App Reviews Analysis

## Project Overview
This project analyzes a synthetic dataset of **2,514 mobile app reviews** collected between 2023â€“2025. The dataset spans **40+ applications** across **24 international languages** and includes reviews from users in **24 countries**. The goal is to extract insights from user feedback and answer key business questions through data analysis and visualizations.

## Tools & Technologies
- **Python**: Data analysis and manipulation  
- **Libraries**: Pandas, NumPy, Matplotlib, Seaborn, Plotly  
- **Jupyter Notebook**: For step-by-step analysis, explanations, and visualizations  

## Project Steps
1. **Data Cleaning**
   - Handle missing values  
   - Fix data types for accurate analysis  
   - Remove duplicates and inconsistencies  

2. **Exploratory Data Analysis (EDA)**
   - Analyze review distribution by app, country, and language  
   - Explore trends in ratings, review length, and sentiment  

3. **Visualization & Insights**
   - Generate charts (bar charts, histograms, heatmaps, interactive plots)  
   - Provide actionable insights to guide business decisions  

4. **Reporting**
   - Submit a Jupyter Notebook with code, visualizations, and explanations  
   - Highlight key findings and trends from the dataset  

## Key Features
- Multi-language and multi-country review analysis  
- Interactive visualizations with Plotly  
- Cleaned and structured dataset ready for further analysis or modeling
